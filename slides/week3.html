  • 物件（Object）是物件

Call by value & call by reference

  • call by value => 傳值, 透過數值傳遞, 複製
  • call by reference => 傳址, 透過參考記憶體位置傳遞, 不複製
  • 物件是call by reference

原型(Prototype)

  • 每一個物件連繫到一個prototype物件, 所有的物件實字都會連繫到Object.prototype

函式(Function)

  • 是JS中的第一級物件（first-class object）[Ref]
  • 提供了作用域(scope)

函式也是物件

  • runtime 動態建立
  • 可以被assign給其他變數、refrence 複製給其他變數、擴充、刪除
  • 可以當作參數傳遞
  • 有自己的屬性或是方法

函式

			//具名函式 named function expression
			var add = fucntion add(a, b) {
				return a+b;
			};

			//不具名函式 anonymous function
			var add = function(a, b) {
				return a+b;
			};

			//函式宣告式function declaration
			function add(a, b) {
				return a+b;
			}
