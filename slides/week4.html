API service, HTTP request and Ajax

API

Application Programming Interface

應用程式接口

顧名思義就是讓應用程式跟應用程式溝通的接口

API是一個很籠統的說法比如WebAPI，其實就是browser提供給web開發人員可以使用JS的一些接口

其他的系統如MIS、ERP...都有可能有他的API來跟別的程式對接

其中，WebService這種類型的API是目前最為常見的一種，會有不同的文件格式（傳輸格式）常見的有JSON, XML...
